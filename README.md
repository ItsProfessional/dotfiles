# My dotfiles

This repository contains dotfiles for my system.

## Installation

* Install `git` and `stow` on your system.
```
pacman -S git stow --needed
```

* Clone the repository.
```
$ cd $HOME
$ git clone git@github.com/ItsProfessional/dotfiles.git
$ cd dotfiles
```

* Create the symlinks
```
$ stow .
```
